## Customization Guide

You can easily show or hide features using boolean properties. For example, to hide the search bar, set `showSearch={false}` (for React) or `:show-search="false"` (for Vue).

- `showSearch`: Controls the visibility of the search input.
- `showPageSize`: Controls the visibility of the page size selector.
- `showPagination`: Controls the visibility of the pagination buttons (Previous/Next).

```tsx
'use client';

import { XStackTable } from '@/LitWrappers';

export default function FullFeaturedTable() {
  const data = [
    { "id": 1, "name": "John Doe", "age": 32, "city": "New York" },
    { "id": 2, "name": "Jane Smith", "age": 28, "city": "London" },
    { "id": 3, "name": "Sam Green", "age": 45, "city": "Tokyo" },
    { "id": 4, "name": "Alice Johnson", "age": 39, "city": "Paris" },
    { "id": 5, "name": "Bob Brown", "age": 51, "city": "Sydney" }
  ];

  const columns = [
    { "key": "id", "label": "ID" },
    { "key": "name", "label": "Name" },
    { "key": "age", "label": "Age" },
    { "key": "city", "label": "City" }
  ];

  return (
    <XStackTable
      columns={columns}
      data={data}
      // Set these to false to hide the controls
      showSearch={true}
      showPageSize={true}
      showPagination={true}
    />
  );
}
```
