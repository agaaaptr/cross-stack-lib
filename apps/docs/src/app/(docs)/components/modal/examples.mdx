## Customization Guide

The modal is highly customizable through properties and slots.

- `open`: A boolean property to control the visibility of the modal.
- `type`: Sets the modal's theme. Accepts `confirmation` (default), `info`, `warning`, or `danger`.

### Slots

You can inject any HTML content into the modal using these named slots:
- `slot="header"`: For the modal title or header content.
- `slot="body"`: For the main content of the modal.
- `slot="footer"`: For action buttons or other footer content.

<div data-framework="react">
```tsx
'use client';

import { useState } from 'react';
import { XStackModal } from '@/LitWrappers';
import { Button } from '@/components/ui/button';

export default function ModalExample() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setIsOpen(true)}>Open Info Modal</Button>
      <XStackModal
        open={isOpen}
        type="info"
        onClose={() => setIsOpen(false)}
      >
        <h2 slot="header">Information</h2>
        <p slot="body">This is an informational message.</p>
        <div slot="footer" style={{ display: 'flex', justifyContent: 'flex-end', gap: '8px' }}>
          <Button variant="outline" onClick={() => setIsOpen(false)}>Cancel</Button>
          <Button onClick={() => setIsOpen(false)}>OK</Button>
        </div>
      </XStackModal>
    </>
  );
}
```
</div>
